# spaceshooterWEB

SpaceShooter 게임 프로젝트를 기반으로 만든 **웹페이지**입니다.  
게임 소개, 플레이 영상, 코드 등을 웹에서 확인할 수 있도록 제작했습니다.

## 주요 기능
- 게임 소개
- 게임 플레이 영상
- 스크립트 버튼들 => 노션페이지로 연결

## 폴더 구조
```
spceshooter.html      # 메인 웹페이지
style.css             # 스타일시트
script.js             # 자바스크립트 파일
.png                  #이미지
```
