# spaceshooterWEB

SpaceShooter 게임 프로젝트를 기반으로 만든 **웹페이지**입니다.  
게임 소개, 스크린샷, 플레이 방법 등을 웹에서 확인할 수 있도록 제작했습니다.

## 주요 기능
- 게임 소개
- 게임 플레이 영상
- 코드 스크립트 버튼들 => 노션으로 연결

## 폴더 구조
```
spaceshooter.html      # 메인 웹페이지
style.css            # 스타일시트
script.js             # 자바스크립트 파일
         # 스크린샷, 아이콘 등 이미지
```
